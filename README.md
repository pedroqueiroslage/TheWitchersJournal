# *The Witcher's Journal*
***
 This project, like **the Witcher's quest catalog**, stores important information
such as name, progress, reward, and status, according to the respective work 
in progress.
***
## Key Features:
Using an interactive CLI menu, you are able to:
   1. **View Quests**: It will show the user reward, progress, status and the quest's name.
   2. **Register Quests**: As you can register all the information about the quest.
   3. **Complete Contracts**: Changing its status.
   4. **Close the Journal**.
***
## Prerequisites:
1. **Python 3.9** or above.
2. Install python's **rich** library.

***
## Setup and Installations:
1. **Clone the Repository**:
```bash
git clone [https://github.com/pedroqueiroslage/TheWitchersJournal.git](https://github.com/pedroqueiroslage/TheWitchersJournal.git)
```
2. **Rich Library installation**:
```bash
pip install rich
```
